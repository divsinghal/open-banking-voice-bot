Azure4Alexa


It's easy to build and host your Alexa Custom Skill with Visual Studio and Azure!

Your Alexa Custom Skill can built as WebAPI project using the excellent AlexaSkillsKit.NET package

Need to host a home page describing your Alexa Custom Skill, along with a Terms of Service and Privacy Policy? No worries, you can place static HTML in the root of your WebAPI project, make use of great client-side frameworks such as Bootstrap, or even add MVC functionality. It's up to you!
